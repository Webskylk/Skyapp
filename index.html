# Sky Circle Pro Invoicing System - Setup Guide ## 1. Overview මෙය HTML, Tailwind CSS, JavaScript සහ Firebase (Database & Auth) භාවිතයෙන් නිර්මාණය කළ සම්පූර්ණ ඉන්වොයිස් පද්ධතියකි. ## 2. Prerequisites - **Firebase Account:** (console.firebase.google.com) - **GitHub Account:** (ස්වයංක්‍රීයව Update වීම සඳහා) - **Netlify Account:** (Hosting සඳහා) ## 3. Firebase Setup (අනිවාර්යයි) 1. **Project එකක් සාදන්න:** "SkyCircle-Invoice" ලෙස. 2. **Authentication:** 'Google' සහ 'Anonymous' (Guest) යන ක්‍රම දෙකම Enable කරන්න. 3. **Cloud Firestore:** 'Test Mode' යටතේ Database එකක් සාදන්න. 4. **Security Rules:** පහත Rule එක Firestore -> Rules වලට දාන්න: ```javascript allow read, write: if request.auth != null; ``` 5. **Settings:** Project Settings වල ඇති Web App Config (Keys) එක index.html හි `firebaseConfig` ස්ථානයට ඇතුළත් කරන්න. ## 4. Key Features Added - **Trial System:** නව පරිශීලකයින්ට දින 30ක Trial එකක් ස්වයංක්‍රීයව ලැබේ. දින 2ක් සහ 3ක් ලෙස උපරිම දින 5ක Extension එකක් User ටම ලබාගත හැක. ඉන්පසු Admin (ඔබ) විසින් Firebase එකෙන් 'expiry' දිනය වෙනස් කළ යුතුය. - **CMS Editor:** Admin Panel එක හරහා App Name, Colors සහ Labels (Travel වෙනුවට ඕනෑම දෙයක්) වෙනස් කළ හැක. - **Recurring Invoices:** මාසිකව හෝ සතිපතා නැවත සිදුවන ගෙවීම් සඳහා විශේෂ පහසුකම. - **Dynamic Fields:** Passport No, Domain Name වැනි ඕනෑම අමතර තොරතුරක් ඇතුළත් කිරීමට 'Add Field' පහසුකම. - **PDF Generation:** ඉතා වේගවත් සහ වෘත්තීය මට්ටමේ PDF නිපදවීම. - **WhatsApp Integration:** 0757219219 අංකයට සෘජුවම සම්බන්ධ වීමේ පහසුකම.
