# Sky Circle Pro (Zoho API Edition) - Setup Guide ## 1. Concept Backend එක ලෙස Zoho Invoice/Books පාවිච්චි කරන අතර, Frontend එක (UI) ලෙස අපේ Custom HTML/Tailwind සයිට් එක පාවිච්චි වේ. ## 2. Zoho Setup 1. [Zoho API Console](https://api-console.zoho.com/) එකට ගොස් 'Server-based Application' එකක් සාදන්න. 2. **Redirect URI** ලෙස ඔයාගේ Netlify URL එක ලබා දෙන්න. 3. Client ID සහ Client Secret ලබාගෙන සුරක්ෂිතව තබාගන්න. ## 3. White-labeling Logic - සියලුම PDF templates අපේ Frontend එකේ සකස් වේ. - Zoho branding ඉවත් කර 'Sky Circle' branding ඇතුළත් කරනු ලැබේ. - Custom fields (Passport, Ticket No) අපේ UI එක හරහා Zoho custom fields වලට Map කරනු ලැබේ. ## 4. Stability Features - Token Refresh Mechanism: ලොගින් එක හැමවෙලේම පවත්වා ගැනීමට. - Error Logging: API එකේ ගැටලුවක් වුවහොත් වහාම හඳුනා ගැනීමට.
